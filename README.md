# 📊 SQL Data Warehouse Project

## 📌 Project Overview
This project demonstrates an End-to-End Data Warehouse solution built using SQL Server.
The goal of this project is to design a modern data warehouse architecture with ETL pipeline,
data transformation, and analytics-ready tables using real-world datasets.

The project integrates multiple source systems (CRM and ERP data) into a centralized
data warehouse using Bronze, Silver, and Gold layers.

This project showcases practical skills required for Data Analyst, SQL Developer,
and Data Engineer roles.

---

## 🏗️ Architecture

The project follows a Medallion Architecture:

- Bronze Layer → Raw data from source systems
- Silver Layer → Cleaned and transformed data
- Gold Layer → Analytics-ready data (Fact & Dimension tables)

Architecture Flow:

Source → ETL → Bronze → Silver → Gold → Analytics

## 🔄 ETL Process

1. Extract data from CRM and ERP sources
2. Load raw data into Bronze layer
3. Clean and transform data in Silver layer
4. Create Fact & Dimension tables in Gold layer
5. Perform data quality checks
6. Optimize queries for performance
